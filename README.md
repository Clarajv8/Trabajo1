Este proyecto presenta el proceso de diseño y desarrollo de la página web de CBA, una revista digital propia centrada en moda sostenible y cultura contemporánea.

El principal objetivo ha sido crear una plataforma que se distancie de la estética minimalista y orgánica predominante en el sector de la sostenibilidad. En su lugar, se propone una identidad visual potente y diferenciada, basada en el uso expresivo del collage digital, una paleta cromática intensa y una tipografía de gran personalidad.

La web busca combinar una navegación intuitiva y funcional con una experiencia visual impactante, capaz de transmitir la voz editorial de CBA y conectar de forma directa con un público joven, consciente y exigente.
